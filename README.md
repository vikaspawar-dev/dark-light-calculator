# 🧮 Dark & Light Calculator

A modern and responsive **Calculator Web App** with a beautiful **Dark & Light Theme Toggle**. Built using **HTML, CSS, and JavaScript**, this project provides a clean UI and smooth user experience.

## ✨ Features

- 🌗 Dark & Light Mode Toggle
- ➕ Basic Arithmetic Operations (+, −, ×, ÷)
- ⌫ Backspace Function
- 🧹 Clear Screen Option
- ⚡ Fast & Responsive UI
- 📱 Mobile Friendly Design

## 🧠 How It Works

- User clicks buttons → values display on screen  
- `eval()` is used to calculate result  
- Theme toggle switches between dark and light mode using class manipulation  

---

## ⚠️ Note

> This project uses `eval()` for simplicity. For production applications, a safer method is recommended.
